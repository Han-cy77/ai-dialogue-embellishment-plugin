一款 Chrome 浏览器插件，用于将简短、口语化的 AI 对话草稿自动扩写成高质量的结构化提示词。

## 功能特性

- **智能提示词优化**：将简短的输入草稿扩写成专业、结构化的提示词
- **一键润色**：在任意网页的输入框旁显示润色按钮，点击即可优化
- **免费模型支持**：使用 SiliconFlow 提供的免费开源模型（Qwen2.5-7B-Instruct）
- **兼容主流 AI 网站**：支持 ChatGPT、Claude 等网页版的输入框
- **框架兼容**：回填时自动触发 React/Vue 框架的数据绑定事件
- **安全加密存储**：API Key 采用 AES-256-GCM 加密算法存储，保障密钥安全

## 安装方法

1. 下载或克隆本项目到本地
   ```bash
   git clone https://github.com/Han-cy77/ai-dialogue-embellishment-plugin.git
   ```

2. 打开 Chrome 浏览器，访问 `chrome://extensions/`

3. 开启右上角的「开发者模式」

4. 点击「加载未打包的扩展程序」，选择本项目文件夹

5. 安装完成，浏览器工具栏会出现插件图标

## 使用说明

### 1. 配置 API Key

1. 点击浏览器工具栏中的插件图标
2. 输入你的 SiliconFlow API Key（[获取地址](https://cloud.siliconflow.cn/)）
3. 点击「保存」按钮（API Key 将被加密存储）

### 2. 使用润色功能

1. 打开任意 AI 对话网站（如 ChatGPT、Claude 等）
2. 在输入框中输入简短的草稿（如"帮我写个贪吃蛇游戏"）
3. 点击输入框右下角的「✨ 润色」按钮
4. 等待优化完成，润色后的提示词会自动回填到输入框
5. 发送即可获得更优质的 AI 回复

## 安全性说明

### API Key 加密存储

本插件采用行业标准加密算法保护您的 API Key：

| 安全特性 | 说明 |
|---------|------|
| 加密算法 | AES-256-GCM（256 位密钥，认证加密） |
| 密钥派生 | PBKDF2 + SHA-256，100,000 次迭代 |
| 随机因子 | 每次加密生成新的 Salt（16字节）和 IV（12字节） |
| 存储方式 | 本地存储（chrome.storage.local），禁用云端同步 |
| 密钥绑定 | 基于扩展 ID 动态生成，无法跨扩展解密 |

### 安全保障

- **非明文存储**：API Key 在存储前经过加密处理
- **本地存储**：数据仅保存在用户本地浏览器，不会上传到任何服务器
- **扩展隔离**：其他网站和扩展无法访问本插件的存储区域
- **动态密钥**：加密密钥基于扩展 ID 动态生成，每个安装实例唯一
